# Python Boids Simulation

## Install Dependencies
  ```
  $ pip install -r requirements.txt
  ```
## Description
Boids algorithm based of the Standford description (https://cs.stanford.edu/people/eroberts/courses/soco/projects/2008-09/modeling-natural-systems/boids.html).

## How to use
  ```
  $ python main.py
  ```
  Run and use the sliders to tweak rules.
